    "# Windows Installation\n",
    "\n",
    "Windows installation is broken up into two categories. The first is for end user installation and the second is for development. The installation guide is based on Windows 10."
   ]
  },
  {
   "cell_type": "markdown",
   "metadata": {},
   "source": [
    "## End User Installation\n",
    "\n",
    "The majority of users will want to simply install the binary distribution of the library. This can be accomplished using pip. If we do not have a binary wheel for your operating system, you must follow the development installation guide.\n",
    "\n",
    "```bash\n",
    "pip install matrixprofile\n",
    "```"
   ]
  },
  {
   "cell_type": "markdown",
   "metadata": {},
   "source": [
    "## Development Installation\n",
    "\n",
    "Due to our usage of Cython and OpenMP for some modules, you are required to install Microsoft Visual C++ Build Tools. This portion of the installation guide will walk you through that."
   ]
  },
  {
   "cell_type": "markdown",
   "metadata": {},
   "source": [
    "### Install Anaconda\n",
    "\n",
    "Anaconda has become the defacto Python distribution for scientific computing and is highly recommended. You may use a base installation of Python without it, but this document does not cover that use case.\n",
    "\n",
    "Download the Anaconda installer for Windows from the [Anaconda website](https://www.anaconda.com/distribution/).\n",
    "\n",
    "Be sure to make this the default Python during the installation."
   ]
  },
  {
   "cell_type": "markdown",
   "metadata": {},
   "source": [
    "### Install Visual C++ Build Tools\n",
    "\n",
    "Go to [https://visualstudio.microsoft.com/downloads/](https://visualstudio.microsoft.com/downloads/) and download the \"Build Tools for Visual Studio\". Scroll down and find it in the section labeled \"Tools for Visual Studio\".\n",
    "\n",
    "![Download Build Tools](install_microsoft_download_build_tools.jpeg)\n",
    "\n",
    "During the installation, be sure to check the box next to \"C++ Build Tools\".\n",
    "\n",
    "![Install C++ Build Tools](install_microsoft_cpp.jpeg)"
   ]

    "### Install matrixprofile for Development\n",
    "\n",
    "Finally, we can clone the matrixprofile source code and install it in development mode. If you wish, you may create a new Anaconda environment for this. Please refer to the [Anaconda documentation](https://docs.conda.io/projects/conda/en/latest/user-guide/tasks/manage-environments.html#creating-an-environment-with-commands) to create a conda environment.\n",
    "\n",
    "Clone the repository using git.\n",
    "\n",
    "Next, launch an Anaconda Powershell.\n",
    "\n",
    "![Anaconda Powershell](install_microsoft_anaconda.jpeg)\n",
    "\n",
    "```\n",
    "cd matrixprofile\n",
    "pip install -e .\n",
    "pip install -r requirements.txt\n",
    "```\n",
    "\n",
    "Once the installation completes, you can try running the tests to verify everything is OK.\n",
    "\n",
    "```\n",
    "pytest tests/\n",
